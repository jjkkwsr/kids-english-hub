## 🎯 學習目標 (Learning Target)

---

## 🔑 核心文法重點與句型 (Grammar Rules & Patterns)

### Some vs. Any

| 句型類型 | 用法 | 範例 |
| :--- | :--- | :--- |
| **肯定句 (Affirmative)** | 用 **some** | There is **some** water. / I have **some** apples. |
| **否定句 (Negative)** | 用 **any** | There isn't **any** water. / I don't have **any** apples. |
| **疑問句 (Question)** | 用 **any** | Is there **any** water? / Do you have **any** apples? |
